# Human-Equipping-Incredible-SEnsor-Interface
HEISEI振り返りセンサー

# 開発環境
- マイコン: ESP32
- センサー: MPU9250 + BMP280
- 基板設計: KiCad 5.1.0
- 言語: Arduino
